Summary of what we learnt every week

Week 1 Summary: Introduction

Lesson was simple, went through an introduction of the module and some information about the module's use (e.g. Web Development)

Week 2 Summary: CSS Basics

  • Learnt different tags to use to separate between sections of text. (Example: "p;/p" tags for paragraphs)
  • Sections include: "header", "body" and sometimes "footer"
  • Introduction and Definition of "CSS"
  • Learnt how to link stylesheet CSS to HTML codes and basic "style" codes (Example: changing fonts/colours)
  • Learnt different selector groups like "ID" and "Class"

Week 3 Summary: More CSS

  • Learnt to create forms using "form", "input" and "label" tags
  • "input" tag can apply to 6 different formats: text, date, color, file, checkbox, radio
  • "validation" tags used to ensure all required fields are filled in, but not necessary for ALL forms
  • Learnt about "Descendant Selector": Specific sections WITHIN a section, can be nested, recommended to use maximum 3
  • Learnt about "Inheritance": When a style is added to an overall group, it applies to all sections within the group (Example: applied to "body" section, any sections under "body" gets same style)
  • Learnt about "Cascades" and "Specificity". If "Specificity" is applied in "Cascade", the more specific group takes precedence
  • Learnt about aligning content using "justify-content", "flex-direction" and "align-items" codes
  • Wireframes

Week 4 Summary: Positioning and Responsive Layouts

  • Responsive images will resize based on screen size, and provide a better performance/design
  • Learnt the tags to use for creating responsive layouts
  • Learnt how to make photos/images resize accordingly automatically
  • Learnt about the scaling system for image sizes
  • Started GITHUB accounts :D
