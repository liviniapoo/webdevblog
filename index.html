Summary of what we learnt every week

Week 1 Summary: Introduction

Lesson was simple, went through an introduction of the module and some information about the module's use (e.g. Web Development)

Week 2 Summary: CSS Basics

  • Learnt different tags to use to separate between sections of text. (Example: "p;/p" tags for paragraphs)
  • Sections include: "header", "body" and sometimes "footer"
  • Introduction and Definition of "CSS"
  • Learnt how to link stylesheet CSS to HTML codes and basic "style" codes (Example: changing fonts/colours)
  • Learnt different selector groups like "ID" and "Class"

Week 3 Summary: More CSS

  • Learnt to create forms using "form", "input" and "label" tags
  • "input" tag can apply to 6 different formats: text, date, color, file, checkbox, radio
  • "validation" tags used to ensure all required fields are filled in, but not necessary for ALL forms
  • Learnt about "Descendant Selector": Specific sections WITHIN a section, can be nested, recommended to use maximum 3
  • Learnt about "Inheritance": When a style is added to an overall group, it applies to all sections within the group (Example: applied to "body" section, any sections under "body" gets same style)
  • Learnt about "Cascades" and "Specificity". If "Specificity" is applied in "Cascade", the more specific group takes precedence
  • Learnt about aligning content using "justify-content", "flex-direction" and "align-items" codes
  • Wireframes

Week 4 Summary: Positioning and Responsive Layouts

  • Responsive images will resize based on screen size, and provide a better performance/design
  • Learnt the tags to use for creating responsive layouts
  • Learnt how to make photos/images resize accordingly automatically
  • Learnt about the scaling system for image sizes
  • Started GITHUB accounts :D

Week 5 Summary: Learning Javascript

For week 5 whitespace week, we used the LinkedIn Learning website to learn about Javascript in Web Development.

  • Learnt about Javascipt itself: Versions, text editors and possible resources
  • Learnt about the uses and basics of variables
  • Learnt about the uses and basics of strings
  • Strings have properties that make coding slightly easier
  • Learnt about the basics of numbers in Javascript
  • There are many types of numbers in code such as floats
  • Learnt about booleans and variable mutability
  • Learnt about the differences between objects and arrays
  • Objects have key-value pairs that must be assigned together when creating the object list
  • Arrays will automatically assign keys to the values that are put into the array
  • Learnt how to use whitespace and comments to make the overall code more readable for both ourselves and others
  • Learnt how to use expressions to check for certain words and phrases in a string.
  • Learnt about how to use simple comparisons (e.g. '==', '===') in codes
  • Learnt how to do math using operators such as addition and division
  • Learnt how to increment values/variables
  • Learnt how to use 'and' (&&) and 'or' (||) to determine whether conditions are true and false
  • Learnt about conditional codes: 'if' and 'switch', and how to use them
  • Learnt about Terse ifs:\ they can make conditional statements shorter into one-lines or they can replace certain areas of conditional statements with symbols, but are a lot more difficult to use;\ they do not use 'true false' to check conditions, instead 'truthy falsey'
  • Learnt how to use 'typeof' to check for the type of a certain code/action used
  • Arrays require a longer code after using 'typeof' as they are identified as Objects with 'typeof' alone
  • Learnt about sequential, enumerative and while loops: their differences and when to use them
  • Sequential loops are 'for' loops that have the conditions right next to the 'for' statements as values/comparisons
  • enumerative loops are 'for' loops that have the conditions next to the 'for' statements as a new variable to be looped in a certain other variable
  • While loops are used for loops that we know approximately how many loops will run
  • Learnt about 'Set and Maps' loops

Week 6 Summary: Recap of Week 5

  • Started to utilise Javascipt on website codes
